# Extrator de E-mails de PDFs (MyBinder + Voilà)

Interface Web para enviar PDFs, extrair e-mails e baixar um arquivo `.docx`.

## Clique aqui para abrir a ferramenta:

[![Binder](https://mybinder.org/)](https://mybinder.org/v2/gh/nucpqsrsp-ai/pdf-email-extractor/HEAD?urlpath=voila/render/app.ipynb)

### Como funciona:
1. Clique no botão acima.
2. Espere o Binder montar o ambiente (pode levar 2–5 minutos no primeiro uso).
3. A interface abrirá como um aplicativo (Voilà).
4. Envie seus PDFs, clique em **Processar** e baixe o arquivo `.docx`.

### Observações importantes:
- O Binder cria ambientes **temporários**, então sempre baixe o arquivo final ao terminar.
- Dependências estão em `requirements.txt`.
- Widgets habilitados pelo script em `binder/postBuild`.
